Staple Dashboard Prod

Production-oriented Staple dashboard for environment setup, wallet-backed actions, pools, pricing, and swaps.

Local Startup Flow

Always open the dashboard through HTTP. The recommended local flow is:

  1. Install dependencies. npm install
  2. Start the local static server from the repository root. npm run start:local
  3. Open the dashboard in the browser. By default it is: http://127.0.0.1:8787

If port 8787 is already occupied, start the server with another port, for example PORT=39001 npm run start:local, then open the matching URL.
